## Overview

Yield Curve Analysis Tool is a standalone HTML dashboard built for fixed income and rates analysis. It gives users a practical way to line up sovereign yield curves, compare present and past snapshots, and interpret curve shifts with central bank policy context layered in.

The interface is tuned for APAC market coverage, where switching between country views makes it easier to assess sovereign curves side by side and judge relative steepness, shape, and movement. With interactive controls and overlayed policy data, it streamlines curve review, historical comparison, and rate relationships without needing a larger software platform.

## How to Use It

1. Select a country tab to switch between APAC sovereign curve views.
2. Use the date picker or date slider to choose the comparison point.
3. Review the chart to see how the curve has moved relative to the selected reference date.
4. Hover over points on the curve to inspect yield, delta, and policy-rate details.
5. Watch the animation controls when you want to step through curve movement over time.
6. Use the movement metrics and slope indicators to support rates analysis and historical comparison.

Example workflow:
- Choose a market
- Compare today against an earlier date
- Check how policy expectations and curve shape differ
- Record the shift in steepness or flattening for your notes
